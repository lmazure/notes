Some people try to classify how far some dev teams are along the AI road.

## Dan Shapiro

[The Five Levels: from Spicy Autocomplete to the Dark Factory](https://www.danshapiro.com/blog/2026/01/the-five-levels-from-spicy-autocomplete-to-the-software-factory/)

| Level | Name | Description (Text from Image) |
|-------|------|------|
| 0 | Spicy Autocomplete | You write every line of code. You use AI like a search engine or for the occasional "tab-to-complete." |
| 1 | Cruise Control | You write the code, but you offload discrete tasks to an AI intern. "Write a unit test for this." "Add a docstring." |
| 2 | Autopilot | You have a junior buddy to hand off the boring stuff to. You're pairing with the AI like a colleague. |
| 3 | Waymo + Safety Driver | You're not a senior developer anymore; that's the AI. You're a manager. You spend your days reviewing diffs. |
| 4 | Robotaxi | You're a PM. You write a spec, argue about it, and then leave for 12 hours to see if the tests pass. |
| 5 | The Dark Factory | A black box that turns specs into software. A place where humans are neither needed nor welcome. |

## Steve Yegge

[Welcome to Gas Town](https://steve-yegge.medium.com/welcome-to-gas-town-4f25ee16dd04#:~:text=you%20should%20locate%20yourself%20on%20the%20chart)

| Stage | Description |
|-------|-------------|
| 1 | Zero or Near-Zero AI: maybe code completions, sometimes ask Chat questions |
| 2 | Coding agent in IDE, permissions turned on. A narrow coding agent in a sidebar asks your permission to run tools. |
| 3 | Agent in IDE, YOLO mode: Trust goes up. You turn off permissions, agent gets wider. |
| 4 | In IDE, wide agent: Your agent gradually grows to fill the screen. Code is just for diffs. |
| 5 | CLI, single agent. YOLO. Diffs scroll by. You may or may not look at them. |
| 6 | CLI, multi-agent, YOLO. You regularly use 3 to 5 parallel instances. You are very fast. |
| 7 | 10+ agents, hand-managed. You are starting to push the limits of hand-management. |
| 8 | Building your own orchestrator. You are on the frontier, automating your workflow. |
